### Data

* `jiang_ght_highlow_pairs_volIDs_346.txt`: is a textfile containing HathiTrust IDs (HTIDs) for each volume reflected in Jiang et al.'s "high" and "low" pairs of aligned Project Gutenberg and HathiTrust Digital LIbrary (HTDL) sentences

* `ht_pg_sents_df_167079_3.csv.zip`: is a CSV file that contains minimal metadata and notable aligned sentences extracted from for works in both HTDL and Project Gutenberg. "Notable" here means that the word error rate (WER) or character error rate (CER) was remarkable high or low. This CSV contains a column that describes if the set of aligned sentences in that row are sourced from high or low WER/CER vales generated by Jiang et al. along with the raw WER/CER scores.
