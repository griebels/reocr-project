# sentence alignment utilities

### Use `sent_match_lev_algo_improved.py`to extract sentences from OCR'd Hathi txt files and align with designated sentences in a CSV. Expects an input CSV to have columns `gsent`, `hsent`, and `hid`. Expects folder containing files in the format `hid.txt`. Best matched sentence is in `match_honed_lev`.
